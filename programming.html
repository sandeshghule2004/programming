📘 What is Programming?

Programming is the process of giving instructions to a computer to perform a specific task. These instructions are written using programming languages.

  • Used to create websites, apps, games, software
  • Examples: C, C++, Java, Python, JavaScript
  • Helps solve real-world problems logically

🌐 HTML Basics

HTML (HyperText Markup Language) is used to create the structure of a webpage.

Basic HTML Structure

<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>

Common HTML Tags

  • <h1> to <h6> – Headings
  • <p> – Paragraph
  • <a> – Link
  • <img> – Image
  • <div> – Container

🎨 CSS Basics

CSS (Cascading Style Sheets) is used to style and design webpages.

CSS Syntax

selector {
    property: value;
}

Example

h1 {
    color: blue;
    text-align: center;
}

Uses of CSS

  • Change colors and fonts
  • Responsive design
  • Layouts (Flexbox, Grid)

⚙ JavaScript Basics

JavaScript is used to make webpages interactive and dynamic.

JavaScript Example

function greet() {
    alert("Hello, Welcome to Programming!");
}

Uses of JavaScript

  • Form validation
  • Dynamic content
  • Games and animations
